# NEON BEAT

4-key neon arcade rhythm game (browser).

## Play online

**https://vox5th-commits.github.io/neon-beat/**

## Play locally

1. Double-click `start.bat`
2. Open http://127.0.0.1:8080
3. Press **START**

## What's new (fun update)

- Hit SFX, EARLY/LATE timing hints
- Life bar (can disable in Settings)
- Scroll speed setting
- Long notes (hold to the end)
- Combo milestones, progress bar
- Local high scores
- Practice mode (loop range + 0.75x)
- Offset calibration mini-tool
- **Import your own music** (mp3/wav/ogg) → auto chart Easy/Normal/Hard
- Longer built-in tracks + section variety (intro / build / drop)
- **Air on the G String** (J.S. Bach) — public-domain composition, original synth arrangement (no commercial recording used)

## Controls

| Key | Action |
|-----|--------|
| D F J K | Lanes |
| Hold | Long notes |
| Esc | Pause |

## Custom music

1. Song select → **내 음악 불러오기**
2. Pick a file you own / have rights to use
3. BPM optional (auto-estimated)
4. Charts are generated and stored in this browser (IndexedDB)

Copyrighted tracks: only use music you have the right to play.

## Settings

Offset, scroll speed, SFX volume, life bar on/off.
